About Ultra Arcade

Ultra Arcade is a Minecraft Network which I am currently developing. Read bellow to see what features I am planning to add or have added

MC Server

The Minecraft part of this server is highly complex. At the base there is Spigot servers running Minecraft 1.8.8. These run a mixture of custom and downloaded plugins. Further up the rungs there is a BungeeCord server which is used by medium sized Minecraft networks and links the Spigot servers together. At some point I might upgrade to RedisBungee which allows me to have multiple BungeeCord server links together

Forums

The other essential counterpart to a Minecraft server is a high quality forums. In my case I used XenForo as a base. I use a wealth of addons to enhance the experince for things like Post Ratings, portal pages and a Wiki. I used a style by the acclaimed Audentio Design Studios which is extremely high quality. I also have done graphic designs for my forums by designing a logo and icons for minigames.

TeamSpeak

Along with the server and forums I am developing a TeamSpeak server which I find to be a higher quality than others by combining aspects of other servers and my own ideas into a single server which includes easy to access links to rules and a decent amount of branding. Along with the forums I have designed most of the graphics used myself.
